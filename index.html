- Portfolio -

Joop is an app for those making new friends. Users journal past adventures and future plans to keep track of who they're spending the most or not enough time with. 5/22/17

Tech: JavaScript, node.js, express, mySQL, bootstrap

Alea is an early stage draft of a social media news app. Users can create a profile, post thoughts and articles to it, view other profiles, comment on other users' posts, and check out the newsfeed for most recent shares. 5/19/17

Tech: Javascript, node.js, express, mySQL, APIs, bootstrap

React News is a MERN stack app. User's can search articles through the NYT's API with a ReactJS-powered interface. Users can also comment on the general discussion board, delete comments, or save other user comments 6/19/17

Tech: Javascript, node.js, React.js, mongoDB, bootstrap

Group Project: Contributed outline, design, and front-end functionality

Moodmash is an app to match your mood with your music. Users select a mood button and are given a corresponding playlist and local concert information on the bands in the playlist. 4/14/17

Tech: Spotify API, JQueryUI, JavaScript, node.js, and bootstrap

Fruits, Veggies, Grains is a long-term nutrition project which functions as an online interactive directory for food group items. Users can learn about a food, it's researched health benefits, and will eventually be recommended recipe ideas (user-auth also coming soon). 3/15/17

Tech: JavaScript, JQueryUI, and bootstrap
